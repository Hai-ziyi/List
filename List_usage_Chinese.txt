List 容器使用指南（中文版）
==============================

1. 概述
-----------
本包提供三种工业级动态数组容器 List，作为 C++ std::vector/deque 与 Python list 的高效替代方案。
- List.hpp          基础版，要求 C++14
- List_pro.hpp      增强版，要求 C++17/20
- List_compatible.hpp 兼容 DevC++ 5.11（GCC 4.9.2，部分 C++14）

所有版本均支持：
  - std::allocator 支持 + EBO 空基类优化
  - 完整迭代器（随机访问、反向、常量）
  - 强异常安全保证
  - 可平凡复制类型高速路径（memcpy/memmove）
  - 调试模式自动自检
  - 内置算法：sort、find、binary_search、rotate、slice 等

2. 文件选择指南
-----------------------
现代编译器（GCC8+、Clang7+、MSVC2019+）    List_pro.hpp
仅支持 C++14                              List.hpp
DevC++ 5.11 / TDM-GCC 4.9.2               List_compatible.hpp

3. 快速上手
--------------
#include "List.hpp"    // 或 List_pro.hpp / List_compatible.hpp

int main() {
    List<int> a;                  // 空列表
    List<int> b(5, 42);           // [42,42,42,42,42]
    List<int> c = {1,2,3,4};      // 初始化列表
    b.push_back(100);             // 尾部添加
    b.pop_back();                 // 删除尾部
    b.insert(b.cbegin(), 0);      // 头部插入
    b.erase(b.cbegin());          // 删除头部
    b.sort();                     // 升序排序
    b.reverse();                  // 反转

    for (auto& x : b) {}          // 范围for遍历
}

front()         首元素，空时抛异常
back()          尾元素，空时抛异常
data()          返回底层数组裸指针

12. 排序顺序枚举
--------------
List<T>::Order::Ascending   升序
List<T>::Order::Descending  降序

13. 非成员函数
------------------------
swap(List&, List&)
erase(List&, val)
erase_if(List&, pred)

14. C++17 类模板推导
-----------------------------
List{1,2,3} 自动推导为 List<int>

15. 分配器支持
---------------------
支持 std::allocator_traits，兼容状态分配器，遵循 POCCA/POCMA/POCS

16. 异常安全
--------------------
所有修改操作提供强异常安全，异常发生时保持原状态不变

17. 线程安全
-----------------
非线程安全，多线程访问需外部同步

18. 调试模式
--------------
定义 NDEBUG 或 LIST_NO_SELF_CHECK 关闭容器自检

19. 性能说明
---------------------
- 扩容因子：1.5 倍
- 可平凡复制类型使用 memcpy/memmove 加速
- 优先使用 noexcept 移动，否则拷贝
- 批量插入前先 reserve 可避免多次扩容
